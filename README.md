# Simple ToDo App With Vue3

```bash
* INSTALL
 npm install

* TEST
 npm run dev

* BUILD
 npm run build

* Vue3 Dev TOOLS
1. Install chrome vue3 js devtools (Beta)
 https://chrome.google.com/webstore/detail/vuejs-devtools/ljjemllljcmogpfapbkkighbhhppjdbg
2. Run
 vue-devtools
```
